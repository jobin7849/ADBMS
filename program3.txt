//1. GRANT:   Give insert and select privilege to user 1 on customer table in the invoice database. Give update and delete privilege to user 2 on all tables in the invoice database. 
mysql> GRANT INSERT, SELECT
    -> ON invoice.customer
    -> TO 'user1'@'localhost';
Query OK, 0 rows affected (0.01 sec)

mysql> 
mysql> GRANT UPDATE, DELETE
    -> ON invoice.*
    -> TO 'user2'@'localhost';
Query OK, 0 rows affected (0.00 sec)

mysql> 

2. REVOKE:   Remove delete privilege of user 2.
mysql> REVOKE DELETE
    -> ON invoice.*
